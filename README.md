# 🎲 Tenzies Game (TypeScript)

A fun and interactive dice game built using React and TypeScript, focused on state management, type safety, and component-based architecture.

---

## 🚀 Overview

Tenzies is a game where the goal is to roll until all dice have the same value. Players can hold dice between rolls to strategically reach the goal.

This project demonstrates strong fundamentals of React along with TypeScript for better scalability and maintainability.

---

## 🛠️ Tech Stack

* React
* TypeScript
* JavaScript
* CSS

---

## 🎮 Game Rules

* Roll the dice
* Click on dice to "hold" their value
* Keep rolling until all dice show the same number
* Win when all dice are held and identical

---

## ✨ Features

* 🎲 Random dice generation
* 📌 Hold/unhold functionality
* 🔁 Re-roll mechanism
* 🧠 Game state management
* 🏆 Win detection logic
* 🔒 Type-safe components using TypeScript
